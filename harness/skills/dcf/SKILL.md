---
name: dcf
description: Valuacion DCF resumida para investigacion financiera profunda.
---

Usa este flujo cuando el usuario pida valorar una empresa por DCF:
1. Identifica ticker, moneda, horizonte y supuestos explicitos.
2. Recolecta ingresos, margen operativo, reinversion, crecimiento, WACC y valor terminal.
3. Presenta rangos conservador/base/agresivo.
4. Declara sensibilidad a WACC y crecimiento terminal.
5. Nunca lo presentes como recomendacion financiera definitiva.
